## 主页

仓库根目录的 `index.html` 是 GitHub Pages 主页，也可以直接双击在本地浏览器打开。主页支持响应式布局、深浅色模式和打印。

如果在线地址尚未生效，可在 GitHub 仓库的 **Settings → Pages** 中，将发布来源设置为 **Deploy from a branch**，选择 `main` 分支和 `/ (root)` 目录。

## 维护规则

- 新旧简历统一放入 `简历/`，旧版使用 `[文件名].历史版本.[扩展名]` 命名，不建立子目录。
- 新公司的准备材料放入仓库根目录的 `求职准备-[公司名]/`。
- 不在公开页面加入未经确认的经历、项目或数据。
- 更新最新简历后，同步核对主页关键数字、任职时间和下载链接。
- 每次提交前更新 `CHANGELOG.md`，记录改动、验证结果和恢复基线。
- 推送前复核暂存内容和目标分支，不使用强制推送。
