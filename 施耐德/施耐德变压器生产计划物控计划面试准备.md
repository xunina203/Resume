# 施耐德变压器生产计划/物控计划面试准备

## 1. 岗位业务背景

施耐德当前变压器业务多为定制化项目，通常更接近 **ETO（Engineer-to-Order，按工程定制）** 模式，而不是标准品的大批量备货模式。

变压器业务的计划管理重点，不只是排产，而是围绕客户项目交付，管理从技术确认、BOM 释放、关键物料采购、生产制造、测试验收到最终发货的完整链路。

核心理解：

> 在变压器 ETO 业务中，计划管理的核心不是单纯排产，而是基于技术冻结节点与关键物料齐套的项目化交付管理。

## 2. 变压器产品基础认知

变压器是电力配电系统中的关键设备，用于电压转换、电能传输、隔离和配电。施耐德相关变压器产品可能应用在工业厂房、数据中心、楼宇、公用设施、能源基础设施等场景。

客户关注点通常包括：

- 电气性能：容量、电压等级、阻抗、温升、损耗等
- 安全可靠性：绝缘、耐压、散热、短路承受能力
- 定制需求：尺寸、接线方式、安装环境、噪音限制、防护等级
- 合规认证：不同国家、行业或客户标准
- 交付周期：项目节点、现场安装窗口、客户投产计划

常见物料类别：

| 类别 | 示例 |
|---|---|
| 核心材料 | 硅钢片/铁芯、铜线或铝线、绝缘材料 |
| 结构件 | 夹件、支架、外壳、底座、紧固件 |
| 电气件 | 套管、分接开关、端子、接线排 |
| 冷却/保护件 | 风机、温控器、散热片、油箱、压力释放装置 |
| 辅助件 | 铭牌、标签、包装材料、螺丝螺母等 |

一台变压器的 BOM 数量差异很大。标准小型产品可能几十到一两百项；中大型或高度定制项目可能几百项甚至上千项。计划和物控的重点不是平均物料数量，而是识别长周期、定制化、不可替代、影响开工和测试的关键料。

## 3. 变压器车间工艺与手机制造的区别

变压器车间通常不会像苹果手机制造那样分成 **SMT** 和 **FATP** 两大段。

- **SMT**：主要用于电子产品的表面贴装，例如手机主板贴片、回流焊、AOI 检测。变压器本体制造一般不以 SMT 为核心。
- **FATP**：Final Assembly, Test & Pack，即最终组装、测试、包装。变压器也有类似阶段，但通常叫总装、出厂测试、包装发运，而不是 FATP。

变压器更可能按以下工序组织：

> 绕线/线圈制造 → 铁芯叠装 → 器身装配 → 浇注或真空干燥/浸漆 → 总装 → 出厂测试 → 包装发运

面试表达：

> 我理解变压器车间不像消费电子那样以 SMT 和 FATP 为主，而是更偏离散制造和项目型制造。计划上要重点关注线圈、铁芯、绝缘处理、总装和测试这些关键工序，尤其是瓶颈工序和测试资源的排程。

## 4. CTO / ATO / MTO / ETO 的区别

| 模式 | 含义 | 特点 |
|---|---|---|
| CTO | Configure-to-Order，按订单配置 | 基于标准模块按规则配置，参数可选，物料和工艺大多标准化 |
| ATO | Assemble-to-Order，按订单装配 | 先备通用件，接单后装配，交期较快 |
| MTO | Make-to-Order，按订单生产 | 接单后生产，设计基本不变，主要是制造层面的按单驱动 |
| ETO | Engineer-to-Order，按工程定制 | 接单后还需要工程设计或深度定制，BOM、工艺、周期都可能变化 |

面试回答：

> CTO/ATO/MTO/ETO 本质是定制深度和订单解耦点不同。ETO 是最前置的按工程定制，计划管理要从项目管理视角出发：先明确技术冻结和关键物料齐套，再做瓶颈产能排程，并通过滚动重排应对变更与插单。目标不是单纯满产，而是按里程碑实现可承诺、可落地的交付。

## 5. ETO 计划逻辑四个核心点

### 5.1 技术冻结点

技术冻结点是指工程设计、图纸、BOM、关键技术参数不再随意变更的时间点。

ETO 产品前期会经历客户需求确认、技术选型、图纸设计、BOM 释放等阶段。如果技术一直变化，采购和生产计划都会反复调整，交期风险会很高。

计划岗位要重点关注：

- 图纸什么时候冻结
- BOM 什么时候释放
- 技术变更是否影响交期、成本和物料
- 变更是否进入 ECN/ECR 流程闭环

### 5.2 关键物料齐套

关键物料齐套是指长周期、不可替代、影响开工或交付的物料按计划到位。

不是所有物料都同等重要。计划/物控要优先管理会卡住项目节点的物料，例如硅钢片、铜/铝导体、套管、分接开关、定制结构件、特殊绝缘件等。

关注重点：

- 哪些料影响开工
- 哪些料影响总装
- 哪些料影响测试或发货
- 哪些料交期长、供应不稳定、替代困难

### 5.3 瓶颈产能锁定

瓶颈产能锁定是提前确认关键工序、关键设备、关键人员或测试资源的产能窗口。

变压器生产可能涉及线圈制造、铁芯装配、总装、浇注/干燥/浸漆、测试等关键环节。某些设备、人力或测试台位可能是瓶颈。

排产不能只看物料是否齐套，还要看瓶颈资源是否有窗口。否则即使物料到了，也可能排不上产。

### 5.4 里程碑交付管理

ETO 订单周期较长，只盯最终交期太晚。需要把订单拆成多个可跟踪节点：

> 需求确认 → 技术冻结 → BOM 释放 → 关键料下单 → 关键料到货 → 开工 → 装配完成 → 测试完成 → 发货

计划管理要通过周滚动或日跟踪机制，识别节点偏差并提前纠偏。

综合表达：

> 这四点其实是 ETO 计划的主线。技术冻结决定 BOM 和采购能不能启动；关键物料齐套决定订单能不能按期开工；瓶颈产能锁定决定即使物料到了能不能排得进去；里程碑管理则是把长周期订单拆成可跟踪节点，提前发现交期风险，而不是等到最后才救火。

## 6. 变压器生产 LT 理解

变压器生产 LT 差异很大，不能给绝对数字。标准小型产品可能几周，定制化中大型变压器通常可能 6-16 周甚至更长。如果包含工程设计、特殊材料采购、客户见证测试或出口认证，整体订单 LT 可能达到 3-6 个月。

| 类型 | 典型 LT 感觉 |
|---|---|
| 标准小型配电变压器 | 2-6 周 |
| 定制干式/油浸式配电变压器 | 6-12 周 |
| 中大型或高度定制变压器 | 12-24 周以上 |
| 特殊项目/进口关键料/客户认证复杂 | 可能 6 个月以上 |

面试表达：

> 我理解变压器 LT 要看产品容量、电压等级、定制程度和关键物料供应情况。标准产品可能几周，但 ETO 项目往往要按项目 LT 来看，通常从技术冻结、BOM 释放、关键料采购、生产、测试到发货，可能是数周到数月。计划上我不会只看制造周期，而会重点拆关键路径，尤其是工程释放、长周期物料和测试资源。

## 7. 生产计划与物控计划岗位侧重点

| 维度 | 生产计划 | 物控计划 |
|---|---|---|
| 核心目标 | 排产可执行、交期达成、产能平衡 | 物料不断线、库存健康、缺料风险可控 |
| 关键指标 | OTD、计划达成率、产能利用率、换线损失 | 缺料率、库存周转、呆滞库存、供应商准交率 |
| 高频场景 | 插单、瓶颈工序、产能不足、计划变更 | 供应延期、MOQ/交期冲突、安全库存设置 |
| ETO 重点 | 项目里程碑、瓶颈资源、测试窗口 | 长周期关键料、BOM 变更、齐套管理 |

## 10. 建议准备的 STAR 案例

### 案例 1：保交付

适用问题：供应商延期、客户插单、订单延期风险。

答题结构：

- S：订单交期紧，关键物料或产能有风险
- T：目标是保证客户交付或最大化降低延期影响
- A：识别关键路径，协调采购/生产/销售，重排优先级，推动替代料或加急方案
- R：用数据说明结果，例如 OTD 提升、延期订单减少、客户投诉降低

### 案例 2：降库存但不影响交付

适用问题：库存优化、物控策略、安全库存。

答题结构：

- S：库存高或呆滞多，但交付不能受影响
- T：降低库存金额，同时保持供应稳定
- A：ABC 分类、关键料安全库存、非关键料按单拉动、呆滞料消耗计划
- R：库存下降、周转提升、缺料率未上升

### 案例 3：变更管理

适用问题：ETO 需求变化、BOM 变更、客户技术变更。

答题结构：

- S：客户或工程变更影响已下单物料/排产计划
- T：控制交期和成本风险
- A：推动变更评审，评估物料、产能、成本、交期影响，更新计划并同步相关部门
- R：变更闭环，交付风险可控，减少返工或呆滞

## 11. 面试总结表达

> 在变压器 ETO 业务里，我会先锁定技术与物料关键路径，再做可执行排程；日常通过缺料预警和齐套管理保障交付，异常时用优先级规则快速重排，目标是在保证 OTD 的同时优化库存结构。
