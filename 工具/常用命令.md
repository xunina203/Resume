# GitHub Copilot CLI 常用命令速查表

下面是 GitHub Copilot CLI 的常用命令和快捷键，适合刚开始使用 Copilot 的小白查阅。

## 常用输入方式

| 命令 / 快捷键 | 作用 | 小白解释 |
| --- | --- | --- |
| `/help` | 查看完整帮助 | 不知道能做什么时，先输入这个 |
| `?` | 查看快速帮助 | 比 `/help` 更简短 |
| `/` | 打开命令提示 | 输入 `/` 后可以看到可用命令 |
| `@` | 引用文件 | 让 Copilot 读取某个文件，例如 `@README.md` |
| `#` | 引用 Issue 或 Pull Request | 让 Copilot 结合 GitHub 上的问题或 PR 回答 |
| `!命令` | 执行终端命令 | 例如 `!git status`，让 Copilot 帮你运行命令 |
| `Shift + Enter` | 输入换行 | 写较长需求时用，不会立刻发送 |
| `Ctrl + C` | 取消当前操作 | Copilot 正在执行时可中断 |
| `Ctrl + C` 两次 | 退出 Copilot CLI | 连按两次退出 |
| `Ctrl + L` | 清屏 | 只清空显示，不删除文件 |

## 会话和上下文

| 命令 | 作用 | 小白解释 |
| --- | --- | --- |
| `/new` | 开始新对话 | 当前话题结束后，用它重新开始 |
| `/clear` | 清空当前会话 | 放弃当前会话内容并重新开始 |
| `/resume` | 切换到历史会话 | 找回之前和 Copilot 的对话 |
| `/rename` | 重命名当前会话 | 给当前会话起一个好记的名字 |
| `/context` | 查看上下文使用情况 | 看 Copilot 还能记住多少当前对话内容 |
| `/compact` | 压缩会话历史 | 对话太长时，用它节省上下文空间 |
| `/copy` | 复制上一条回复 | 快速复制 Copilot 刚刚给出的内容 |
| `/search` | 搜索当前会话 | 在历史对话里找关键词 |
| `/share` | 导出或分享会话 | 可把会话保存成文件或 GitHub gist |

## 项目和代码相关

| 命令 | 作用 | 小白解释 |
| --- | --- | --- |
| `/init` | 初始化项目说明 | 让 Copilot 为当前仓库生成或配置说明文件 |
| `/ide` | 连接 IDE 工作区 | 让 Copilot 结合编辑器里的项目工作 |
| `/diff` | 查看当前改动 | 看 Copilot 或你自己改了哪些文件 |
| `/review` | 运行代码审查 | 让 Copilot 检查代码里的潜在问题 |
| `/security-review` | 运行安全审查 | 专门检查安全风险 |
| `/pr` | 处理当前分支的 Pull Request | 查看或操作当前分支相关 PR |
| `/lsp` | 管理语言服务器 | 帮助 Copilot 更懂代码结构 |
| `/terminal-setup` | 配置终端多行输入 | 如果 `Shift + Enter` 不好用，可运行这个 |

## 模型、代理和自动化

| 命令 | 作用 | 小白解释 |
| --- | --- | --- |
| `/model` | 选择 AI 模型 | 不确定时可以选择 `auto` |
| `/agent` | 浏览和选择可用代理 | 不同代理适合不同任务 |
| `/tasks` | 查看和管理任务 | 查看 Copilot 正在跑的子任务或命令 |
| `/delegate` | 交给 GitHub 创建 PR | 让 Copilot 在 GitHub 上继续完成并提交 PR |
| `/fleet` | 开启并行代理模式 | 适合复杂任务，让多个代理并行工作 |
| `/autopilot` | 开关自动驾驶模式 | 也可以理解为 auto pilot，让 Copilot 更自动地推进任务 |
| `/skills` | 管理技能 | 启用更强的专项能力 |
| `/mcp` | 管理 MCP 配置 | 连接外部工具或服务 |
| `/plugin` | 管理插件 | 扩展 Copilot CLI 的功能 |

## 权限和工作目录

| 命令 | 作用 | 小白解释 |
| --- | --- | --- |
| Permissions | 权限相关功能分类 | 这不是直接输入的命令，下面这些 `/allow-all`、`/add-dir`、`/list-dirs` 都属于权限相关 |
| `/cwd` | 查看或切换当前工作目录 | 确认 Copilot 正在哪个文件夹里工作 |
| `/add-dir` | 添加可访问目录 | 允许 Copilot 读取更多文件夹 |
| `/list-dirs` | 查看已允许访问的目录 | 看 Copilot 当前能访问哪些目录 |
| `/allow-all` | 开启全部权限 | 也可以理解为 permission allow all，方便但权限较大，新手谨慎使用 |
| `/reset-allowed-tools` | 重置工具权限 | 权限配置混乱时可恢复 |

## 账号、设置和诊断

| 命令 | 作用 | 小白解释 |
| --- | --- | --- |
| `/login` | 登录 Copilot | 第一次使用或登录失效时使用 |
| `/logout` | 退出登录 | 切换账号时使用 |
| `/settings` | 打开或查看设置 | 调整 Copilot CLI 配置 |
| `/version` | 查看版本 | 排查问题时常用 |
| `/update` | 更新 Copilot CLI | 升级到最新版 |
| `/diagnose` | 诊断当前会话日志 | 出问题时让 Copilot 帮你分析 |
| `/feedback` | 提交反馈 | 向 GitHub 反馈问题或建议 |
| `/theme` | 设置颜色主题 | 调整终端显示风格 |
| `/memory` | 查看或管理记忆功能 | 控制 Copilot 是否跨会话记住信息 |

## 新手推荐用法

| 目标 | 推荐输入 | 说明 |
| --- | --- | --- |
| 不知道怎么开始 | `/help` | 先看 Copilot CLI 能做什么 |
| 让 Copilot 看某个文件 | `@文件名 请帮我解释这份文件` | 例如：`@README.md 请用中文解释` |
| 让 Copilot 执行命令 | `!git status` | 前面加 `!` 就是运行终端命令 |
| 允许 Copilot 自动使用更多权限 | `/allow-all` | 对应 permission allow all，适合你信任当前项目时使用 |
| 找回之前的对话 | `/resume` | resume 表示继续之前的会话 |
| 让 Copilot 更自动推进任务 | `/autopilot` | auto pilot 表示自动驾驶模式 |
| 检查当前修改 | `/diff` | 看当前目录里改了什么 |
| 让 Copilot 审查代码 | `/review` | 提交前检查问题 |
| 对话太长 | `/compact` | 压缩历史，让 Copilot 保持理解能力 |
| 开启新任务 | `/new` | 避免旧话题干扰新任务 |
